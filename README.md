## 🧑‍💻 Обо мне

Я разработчик с **2018 года**. Проектирую и создаю backend-платформы, security-инструменты, developer tooling и инженерные системы на стыке **Node.js, TypeScript, AI, инфраструктуры и прикладной инженерии**.
