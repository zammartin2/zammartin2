## 📦 Коммерческие проекты (закрытые)
### 🧠 UAIP — Когнитивная инженерная платформа
**Роль:** Архитектор платформы, ведущий разработчик
**Описание:**
Платформа нового поколения для инженерных расчётов, симуляций и проектирования. Объединяет AI-рассуждение, физические симуляции, XR-визуализацию и интеграцию с CAD/PLM в единой когнитивной среде.
**Ключевые возможности:**
- 🧬 **Научные вычисления:** биология, химия, материаловедение, квантовая физика
- 🔧 **Инженерный пайплайн:** CAD-интеграция → симуляции → цифровые двойники → AR-сборка
- 🧠 **Когнитивный AI:** объяснимые решения, генерация гипотез, адаптивное планирование
- 👓 **XR-интерфейсы:** WebXR, AR/VR для инженерных задач
- 🔗 **Интеграции:** SolidWorks, Fusion360, SAP, Teamcenter, поставщики компонентов
**Стек:** TypeScript, NestJS, Go, Vue 3, Three.js, PostgreSQL, Kafka, Kubernetes

  

### ✈️ Aerounion (CTZ + Office)
Коммерческие проекты для авиации (GPS-трекинг, CAN-шина, SOS) и управления офисом (ERP-система).  
**Стек:** NestJS, TypeScript, PostgreSQL (Prisma), Redis, WebSocket, Docker, Ollama
